Web Development Philosophy

My development philosophy adheres to the KISS principle. I use the right set of tools for the goal of a project, and am constantly reevaluating my work throughout the development process to ensure the UX is informative and fast. I am adept at adapting to new languages and frameworks and strive to choose the optimal method for each individual project.

Here is an overview of my development process:

  • Data extraction via APIs and python pandas and construction of data storage system
  • Engineer an efficient data delivery network using frameworks like Flask based on the rate of data change
  • Develop frontend using Bootstrap and JavaScript libraries like jQuery, Google Maps and Charts

Georgia Family Connection Partnership
Custom templating solutions and data visualization products

Georgia Family Connection Partnership is a public/private partnership that supports 159 Family Connection county collaborative organizations across the state and serves as a resource to state agencies in improving conditions for Georgia's children and families. I develop and maintain online templating systems that allow the local member to input custom text into a printable PDF profile of their county or other geographic area. I interface with the Organization Management System (OMS) to pull data in real time to allow for the public to search for their local representative of the organization. I also use information entered into the OMS to create templates for county level organization members to help them better manage their local collaborative.

Online agricultural management software

For my Master's research, I incorporated my backgroud in geospatial analytics to Precision Agriculture.
